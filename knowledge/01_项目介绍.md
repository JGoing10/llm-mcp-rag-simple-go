# 项目介绍

LLM-MCP-RAG Simple 是一个用 Go 编写的轻量级智能代理示例，提供：
- 与兼容 OpenAI 的聊天 API 交互（支持流式输出与工具调用）
- 基于 MCP 协议调用外部工具（示例：计算器服务器）
- 使用向量检索的 RAG 能力（内存向量库 + 余弦相似度）

## 主要组件
- `agent/agent.go`：封装对话编排、RAG 检索、MCP 工具调用与重试
- `chat/openai.go`：OpenAI 兼容聊天客户端，支持工具调用与历史管理
- `embedding/embedding.go`：文本向量化与相似度检索
- `vectorstore/vectorstore.go`：内存向量存储，支持并发安全与相似度计算
- `mcp/client.go`：MCP 客户端，会话与工具发现、调用
- `config/config.go`：按 `.env` 加载并校验配置
- `main.go`：程序入口，提供交互式命令行体验

## 适用场景
- 企业知识库问答、技术文档检索
- 工具增强的自动化助手（文件/网络/计算等）
